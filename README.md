# Einfache Lebensmittelverwaltung
## Idee
Ein Ipad das am Kühlschrank hängt, womit jeder auf das Programm zugreifen kann.

Es soll die Übersicht vereinfachen, was zurzeit im Haus an Lebensmitteln ist. So können alle sehen was einzukaufen ist und was shon da ist.

Das Programm soll so einfach wie möglich gehalten werden damit meine Familie es auch benutzt.

## Zusatz
Die Lebensmittel sind nicht alle an einem Ort deswegen kann man nachträglich angeben wo sich diese befinden.

Verbrauchte Lebensmittel können direkt wieder auf die Einkaufsliste gestz werden. 

Eine extra to-do liste für Aufgaben im Haushalt

...

## Weitere Ideen
Der Button Ideen zum Kochen soll einen Prompt mit den Elementen aus der Lebensmittelliste in eine KI geben die drei kochrezepte vorschlägt.

Es soll eine Funktion geben die handschriftliche Einkaufslisten einließt und hinzufügt.

Die erstellte Einkaufsliste PDF soll direkt gedruckt werden können.
